# Create: Cartels

## Descripton
This mod adds many cartel related items and features such as the ability to distribute and sell your product

## Developing
### Using IntelliJ IDEA
If using IntelliJ as your IDE you will need the Minecraft development plugin and a copy of this source code, download with 
```
git clone https://github.com/coderwizard24/create-cartels/
```
simply open the folder as a project in intelliJ, and it should automagically build the gradle project

if you have any problems or simply want to request a feature, open an issue and i'll work on fixing quickly
